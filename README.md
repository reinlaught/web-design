## Мета та Завдання
**Мета:** ознайомитись із засобами мови Javascript та навчитись їх застосовувати для побудови Web-інтерфейсу користувача.  
**Завдання:** розробити функціональність для статичних сторінок Web-додатку першої лабораторної роботи із використанням архітектурного шаблону MVC. В якості сховища даних використати `localStorage`. Заборонено використання JS-фреймворків (ReactJS, AngularJS, VueJS тощо).

## Технології
* **HTML5 та CSS3** — семантична розмітка та базова стилізація.
* **Tailwind CSS** — фреймворк для швидкої побудови адаптивного інтерфейсу.
* **Vanilla JavaScript (ES6+)** — реалізація всієї клієнтської логіки без сторонніх бібліотек.
* **MVC (Model-View-Controller)** — патерн організації коду для головної робочої сторінки.
* **Web Storage API (localStorage)** — імітація бази даних для збереження користувацьких профілів та сесій.

## Опис додатка: CalcApp
**CalcApp** — це сучасний браузерний веб-калькулятор. У другій лабораторній роботі додаток отримав повноцінну програмну логіку та спеціалізований **"Режим програміста"**.

### Основний функціонал:
**Калькулятор** (`index.html`): 
   * Розроблений за шаблоном MVC.
   * Виконує базові математичні обчислення (звичайний режим).
   * Режим програміста: конвертація чисел між десятковою (DEC), шістнадцятковою (HEX), вісімковій (OCT) та двійковою (BIN) системами.
   * Повна підтримка керування з клавіатури ПК, історія операцій та розумне форматування великих результатів.
**Вхід до сайту** (`login.html`) та **Реєстрація** (`register.html`): 
   * Робоча система авторизації зі збереженням даних та перевіркою паролів через `localStorage`.
**Профіль користувача** (`profile.html`): 
   * Динамічний вивід особистих даних залогіненого користувача (ім'я, email, вік, стать) та генерація аватарки за ініціалами. Наявний функціонал редагування профілю.
**Про додаток** (`about.html`) — опис проєкту та оновленого стеку технологій.
