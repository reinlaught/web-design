## Завдання
Розробити статичні Web-сторінки додатка та перевірити коректне їх відображення на екранах різного розміру (адаптивна верстка). Програмування функціональності на даному етапі не передбачено.

## Технології
* **HTML5** — семантична розмітка.
* **CSS3** — стилізація.
* **Tailwind CSS** — фреймворк для швидкої побудови інтерфейсу та забезпечення адаптивності.

## Опис додатка: CalcApp
**CalcApp** — це сучасний веб-калькулятор із чистим та інтуїтивно зрозумілим інтерфейсом. Додаток підтримує роботу на десктопах, планшетах та смартфонах.

### Перелік обов’язкових сторінок:
1. **Калькулятор** (`index.html`) — головна робоча сторінка з інтерфейсом калькулятора.
2. **Вхід до сайту** (`login.html`) — форма авторизації.
3. **Реєстрація** (`register.html`) — форма створення нового акаунта.
4. **Профіль користувача** (`profile.html`) — сторінка з даними користувача у табличному вигляді.
5. **Про додаток** (`about.html`) — опис проєкту та використаного стеку технологій.
