CalcApp Web

CalcApp — це сучасний, адаптивний веб-калькулятор, розроблений в рамках лабораторної роботи №1 та лабораторної роботи №2 з дисципліни "Web-дизайн". Додаток дозволяє виконувати базові математичні операції та має зручний користувацький інтерфейс, оптимізований як для десктопних, так і для мобільних пристроїв.

Технологічний стек:

  • HTML5 (Семантична розмітка сторінок)
  • CSS3 (Стилізація елементів)
  • Vanilla JavaScript (ES6+) (Реалізація функціональності сайту)
  • Шаблон MVC (Model-View-Controller патерн)
  • Tailwind CSS (Фреймворк для швидкої верстки та адаптивності)
  • Web Storage API (localStorage як сховище даних)
